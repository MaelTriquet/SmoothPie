# SwiftPie 🍰

SwiftPie is a lightweight Android launcher focused on speed, gesture-based interaction, and a radial (pie) menu UI.

Instead of navigating through grids of icons, SwiftPie lets you launch apps through a fast and fluid circular interface designed for efficiency and minimalism.



## FEATURES

- Fast app launching via radial (pie) menu
- Gesture-driven navigation
- Customizable menu structure
- Minimal and distraction-free UI
- Built entirely in Kotlin



## CONCEPT

SwiftPie replaces the traditional home screen with a circular launcher interface:

- Apps are organized in a pie menu
- Interaction is based on direction and movement
- Designed to reduce friction and improve speed
